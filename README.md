# DevOps Engineer 7주 완성 과정

## 📚 과정 소개
현대 소프트웨어 개발에서 필수가 된 DevOps 문화와 기술을 체계적으로 학습하는 7주 집중 과정입니다.

## 🎯 학습 목표
- DevOps 문화와 철학 이해
- Docker 컨테이너 기술 마스터
- Kubernetes 오케스트레이션 활용
- CI/CD 파이프라인 구축
- Infrastructure as Code 실무 적용
- 최신 DevOps 트렌드 습득

## 📅 과정 구성
- **총 기간**: 7주 (280시간)
- **주당 시간**: 40시간 (8시간/일 × 5일)
- **세션 구성**: 50분 수업 + 10분 휴식

## 🔄 학습 방식 개편
- **Week 1**: 이론 중심 (80% 이론 + 20% 개념 예시)
- **Week 2+**: 실습 중심 (40% 이론 + 60% 실습)
