🔍 컨테이너 보안 취약점 스캔 실습

실습 목표

이 웹사이트는 의도적으로 취약한 컨테이너 이미지를 사용하여 구축되었습니다.

Trivy를 사용해서 취약점을 찾고 수정해보세요!

🚨 포함된 취약점들

  • Critical: Nginx 1.14.0 (CVE-2019-20372)
  • High: 구버전 Debian 패키지들
  • Medium: Root 사용자 실행
  • Low: 불필요한 패키지 포함

📋 실습 단계

  1. 취약한 이미지 빌드: docker build -f Dockerfile.vulnerable -t vulnerable-web .
  2. Trivy로 스캔: trivy image vulnerable-web
  3. 취약점 분석 및 Critical 취약점 확인
  4. 보안 강화된 이미지 빌드: docker build -f Dockerfile.secure -t secure-web .
  5. 다시 스캔해서 개선 확인: trivy image secure-web

✅ 성공 기준

Critical 취약점이 0개가 되면 성공!

High 취약점도 현저히 줄어들어야 합니다.
