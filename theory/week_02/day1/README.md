## 📚 이론 강의 (2.5시간 - 50분×3세션)

- **Session 3**: 네트워크 보안과 방화벽 설정 (보안 강화)

- Custom 네트워크 생성과 고급 설정
- 네트워크 드라이버별 특성과 선택 기준

- DNS 기반 서비스 디스커버리
- 로드 밸런싱과 헬스 체크
- 네트워크 별칭과 링크 활용

### Session 3: [네트워크 보안 & 방화벽 설정](./session_3.md)
