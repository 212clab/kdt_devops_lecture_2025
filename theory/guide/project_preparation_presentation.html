📝 DevOps 프롬프트 템플릿

역할: 당신은 10년 경력의 DevOps 아키텍트입니다.

상황:
- 마이크로서비스 기반 웹 애플리케이션
- 사용자 1만명, 일일 트래픽 10만 요청
- AWS 클라우드 환경, 예산 제약: 월 $500

요청:
1. Kubernetes 기반 아키텍처 설계
2. 비용 최적화 방안
3. 확장성 고려사항

출력 형식: 아키텍처 다이어그램 (Mermaid), 구성요소별 설명, 비용 분석
